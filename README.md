Offline-first journal PWA using **IndexedDB** + vanilla HTML/CSS/JS.

## Run locally
Use any static server:
- VS Code Live Server
- `python -m http.server`

Then open:
- http://localhost:8000

## Install as PWA
Open in mobile browser (Chrome/Safari iOS) and choose **Add to Home Screen** (or Install).

## Backup
- EXPORT creates a JSON file with entries + projects
- IMPORT restores from that JSON

## Paper styles
- Default: utility grid background (Field Notes-inspired)
- Optional ruled mode: add `paper-ruled` class to `<body>` in index.html
